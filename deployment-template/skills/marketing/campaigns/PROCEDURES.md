# Campaign Management Procedures

## Campaign Types

### Product Launch
```
Timeline: 4-6 weeks
Phases:
1. Teaser (2 weeks out) - hints, curiosity
2. Announcement (launch day) - full reveal
3. Push (week 1) - social proof, urgency
4. Last chance (week 2) - final CTAs
```

### Seasonal/Event
```
Timeline: 2-4 weeks
Phases:
1. Awareness (2 weeks out)
2. Build-up (1 week out)  
3. Event day - heavy push
4. Follow-up - recap, next steps
```

### Evergreen
```
Ongoing nurture:
- Weekly/biweekly touchpoints
- Mix of value + soft pitch
- Segment by engagement
```

## Multi-Channel Coordination

| Channel | Role | Timing |
|---------|------|--------|
| Email | Primary driver | Day 1, 3, 7 |
| Social | Awareness + social proof | Daily during campaign |
| Landing page | Conversion | Always live |
| Retargeting | Catch drop-offs | Day 2+ |

## Campaign Checklist

### Pre-Launch
- [ ] Goal defined and measurable?
- [ ] Audience segmented?
- [ ] All assets created?
- [ ] Landing page live and tested?
- [ ] Tracking/UTMs set up?
- [ ] Team briefed?

### During
- [ ] Monitoring performance daily?
- [ ] Responding to engagement?
- [ ] Adjusting based on data?

### Post
- [ ] Results documented?
- [ ] Learnings captured?
- [ ] Follow-up sent to engaged leads?
