## üì¶ Size Management Tips

1. **Use Low-Poly models** - Search for "low poly" versions
2. **Compress textures** - Reduce from 4K to 1K where possible
3. **2D over 3D for backgrounds** - Images use less space
4. **Reuse assets** - Same server rack model, different positions
5. **Check file sizes before importing** - Most Sketchfab models show size
