# SEO & Analytics

## Tools

```
web_search(query)   → Check rankings, find keywords
web_fetch(url)      → Analyze page content
```

## Quick wins for SMBs

1. **Google Business Profile** — Priority #1 for local businesses
   - Claim and verify
   - Complete all fields
   - Add photos weekly
   - Respond to ALL reviews

2. **On-page basics**
   - Title tag: primary keyword + brand
   - Meta description: 150 chars with CTA
   - H1: one per page, includes keyword
   - Local: city name in content

## Keyword research

```python
# Check what ranks
web_search("plomero CDMX")  # See who's ranking

# Find questions people ask
web_search("cómo elegir plomero")

# Check competitor keywords
web_fetch("competitor-url.com")  # Look at their title/content
```

## Analytics checklist

Before analyzing, verify:
- [ ] GA4 installed and collecting?
- [ ] Conversion goals defined?
- [ ] Enough data? (50 visits = not significant)

## Gotchas

- SEO is long game — set expectations: 3-6 months for results
- Never keyword stuff or use black-hat techniques
- Local SEO (Google Business Profile) >> website SEO for most SMBs
