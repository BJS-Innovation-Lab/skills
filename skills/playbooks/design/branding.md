# Branding

## Skills to Use

| Skill | When |
|-------|------|
| `media-generation` | Generate logo concepts, brand graphics |
| `figma-api` | Extract brand assets, design tokens |
| `web_search` | Research competitor branding |
| `firecrawl` | Scrape brand guidelines from sites |

---

## What I can do

- Logo **concepts** (AI-generated starting points)
- Color palettes with hex codes
- Font pairing recommendations
- Brand guidelines document
- Mood boards

## What needs a professional

- Final production-ready logo
- Trademark search
- Print-ready files (CMYK, vectors)

## Brand discovery questions

1. "¿Ya tienen algo de branding?" (don't start from scratch if they have something)
2. "¿Qué industria?" (law firm ≠ taco shop)
3. "¿Quién es el cliente ideal?"
4. "3 palabras que describan cómo quieren ser percibidos"

## Brand guidelines doc

```markdown
# Manual de Marca: [Client]

## Logo
- [Usage rules]
- [Minimum size]
- [Clear space]

## Colores
[Palette table]

## Tipografía
- Títulos: [Font]
- Cuerpo: [Font]

## Voz y tono
- [How they speak]
- [Words to use]
- [Words to avoid]

## Do's and Don'ts
- ✅ [Do]
- ❌ [Don't]
```

## Gotchas

- Always present 3 directions — let client react
- Research competitors BEFORE designing
- Branding is emotional — if they don't like it, try completely different direction
