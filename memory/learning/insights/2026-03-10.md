## Multi-Agent Context via Polling Pattern

**Context:** Working on Frontier Lab multi-agent collaboration with Bridget. Needed agents to see each other's messages for meaningful conversation.

**Learning:** Context-aware multi-agent systems don't need realtime subscriptions. A simple polling pattern works excellently:
- Agent receives webhook trigger → fetches last N messages → responds with full context
- No RAM bump needed, no persistent connections, 90% of realtime value at 0% extra cost
- Proof: Scout referenced Vulki's Harvard proposal message, showing she could see context

**Stakes:** high - This pattern could be applied to many multi-agent systems instead of expensive realtime infrastructure

## Leapfrog AI Research Insight

**Context:** Developing Harvard Ingenuity Award application for VULKN grants project.

**Learning:** "Leapfrog AI" is a powerful framing for AI agent businesses targeting developing markets:
- Paper → AI (skipping computers entirely)  
- Like mobile phones leapfrogging landlines in Africa
- Like M-Pesa leapfrogging traditional banking
- Novel research question: "What happens when people who've never used computers for work suddenly get AI?"
- Academic backing: IMF data shows 26% AI exposure in developing countries vs 40% globally

**Stakes:** high - This research angle differentiates AI agent companies from "another chatbot" and opens grant/academic opportunities
