# Insights - March 8, 2026

## Research Synthesis Methodology for AI Literature
**Context:** Created Frontier Lab proposal by analyzing 9 arXiv papers on Moltbook agent social behavior
**Learning:** Effective research synthesis pattern: (1) systematic paper collection, (2) extract key metrics/findings, (3) identify patterns across studies, (4) translate to design principles, (5) create research-backed proposals. The Moltbook literature provided concrete evidence for agent social behavior problems.
**Stakes:** medium (improves future research quality and credibility)

## Three-Layer Cloud Agent Architecture
**Context:** Documenting Scout deployment and cloud agent setup
**Learning:** Clean separation works: (1) clawdbot-railway-template = Docker/OpenClaw/setup, (2) vulkn-field-template = skills pulled fresh on restart, (3) [agent]-brain = identity/memory restored. Skills don't need to be baked in - they're fetched via git pull.
**Stakes:** medium (enables scalable cloud deployments)

## Agent-Driven Memory Refresh > Script-Based
**Context:** Replaced memory-load.cjs script with agent-driven memory refresh cron job
**Learning:** Scripts can't understand context - they search for meaningless queries like "periodic refresh" and output noise. Agents read and comprehend, know what matters, output signal. Agent-driven automation is more intelligent than script-based.
**Stakes:** medium (improves memory system quality)

## Markdown-First Documentation Process
**Context:** Created cloud agent setup guide, Bridget preferred Markdown for editing
**Learning:** Edit in Markdown first (easier to track changes, make surgical edits), convert to Word only when finalized for external sharing. Allows iterative improvement without formatting overhead.
**Stakes:** low (improves documentation workflow)
