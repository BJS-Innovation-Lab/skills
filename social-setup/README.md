# Social Setup Skill

Set up social media accounts and API integrations for clients.

## Quick Start

1. Read `SKILL.md` for full procedures
2. Use `templates/new-client-checklist.md` for each new client
3. Store credentials in `~/.config/social/{platform}.json`

## What This Skill Covers

- Creating business accounts on LinkedIn, Facebook, Instagram, TikTok, WhatsApp
- Setting up API access for automated posting
- Connecting accounts to the agent
- Token refresh and maintenance

## Dependencies

- mac-use skill (for browser automation fallback)
- marketing-module skill (for content creation)
- Client brand profile (from intake)
