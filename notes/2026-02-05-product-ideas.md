## 2. Consumer App ("Truth Check" / Working Title)

### Target Audience
- 20-somethings (avg age ~22)
- Use cases: Checking if boyfriend/girlfriend is lying, friend conversations
- Appeal: Fun, novelty, almost like a toy
- Low barrier to entry - grab and use

### Core Features

**Recording:**
- Record in-person conversations
- Record during phone calls
- Import voice messages
- (Future) In-person service interactions (plumber, salesperson, etc.)

**Analysis Output:**
- Fun graphic interface with conversation transcript
- Interactive - highlights less truthful sections
- Tap/pop into sections for deeper analysis
- Cool, fun, bright graphics BUT also minimal/clean

**User Profiles:**
- Create profiles for people (boyfriend, friend, etc.)
- Ongoing analysis tied to each person
- Track patterns over time
- Build behavioral profiles (like Subject feature in prototype)
