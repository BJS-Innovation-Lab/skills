The Vision

Not a party game. A personal tool.

Think: Voice Memos meets Find My meets... trust.

A clean, minimal utility that 20-somethings use to get a "read" on conversations. Low-key. Personal. Almost like checking the weather - quick, useful, part of your routine.

UI Direction

Color Palette

  • Primary: Clean white/light gray
  • Accent: One subtle color (soft blue? mint? coral?)
  • Text: Dark gray, not pure black
  • Highlights: Soft, not alarming (amber/yellow, not red)

Typography

  • San Francisco (iOS) / Roboto (Android)
  • Clean, readable, lots of breathing room
  • Big headers, comfortable text size

Components

  • Minimal chrome
  • Floating action button for record
  • Card-based layout for recordings
  • Smooth animations, nothing jarring
