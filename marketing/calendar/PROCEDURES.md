## Planning Process

### Weekly Planning (Every Friday/Monday)

```
1. Review last week's performance (check learnings.md)
2. Check upcoming dates/events
3. Assign content themes to each day
4. Create draft entries in calendar
5. Review with owner
```

### Monthly Planning (First of Month)

```
1. Set monthly themes (product, education, stories, etc.)
2. Identify key dates (holidays, launches, events)
3. Allocate content types across weeks
4. Set goals (X posts per platform)
5. Review with owner
```

---

## Content Mix Guidelines

### Recommended Weekly Mix

| Content Type | Frequency | Purpose |
|--------------|-----------|---------|
| Thought leadership | 2x/week | Build authority |
| Customer stories | 1x/week | Social proof |
| Educational | 1x/week | Provide value |
| Behind-the-scenes | 1x/week | Humanize brand |
| Promotional | 1x/week MAX | Drive action |

### The 80/20 Rule

- **80%** Value (educate, entertain, inspire)
- **20%** Promotional (sell, CTA, offers)

---

## Platform Cadence

| Platform | Recommended | Minimum |
|----------|-------------|---------|
| LinkedIn | 3-5x/week | 2x/week |
| Facebook | 1x/day | 3x/week |
| Instagram | 1x/day + stories | 3x/week |
| Twitter/X | 3-5x/day | 1x/day |
| Email | 1x/week | 2x/month |

---

## Calendar Review Checklist

Before approving the week's calendar:

- [ ] Mix of content types (not all promotional)?
- [ ] Voice consistent with brand profile?
- [ ] Key dates covered?
- [ ] Enough lead time to create?
- [ ] Images/graphics assigned?
- [ ] Owner approved?

---
